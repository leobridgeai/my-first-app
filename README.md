# MENA Readiness Assessment

Go-to-Market Readiness Diagnostic for SaaS Companies Entering the Middle East & North Africa. A branded tool by **Bridge AI**.

## Quick Start

```bash
npm install
npm run dev
```

## Features

- Interactive 18-question assessment across 6 categories
- Real-time scoring with weighted calculations
- Readiness tier classification (Market Ready / Conditionally Ready / Not Ready)
- Auto-populated gap tracker roadmap
- PDF export via print
- Responsive design for desktop and tablet

## Tech Stack

React 19 + Vite 7. See [CLAUDE.md](./CLAUDE.md) for full documentation.
